**Travel Explorer** is a multi-page, responsive website built with **HTML, CSS, and JavaScript**, designed to showcase travel destinations, hidden gems, cost comparisons, and a functional contact form. The project demonstrates core web-development skills, including layout design, multimedia usage, form handling, accessibility features, and user-friendly UI elements.

- Responsive & modern layout
